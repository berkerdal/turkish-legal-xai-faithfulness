# Data

The dataset is not stored in this repository. It is public and available on Hugging Face:

- `icgcihan/Turkish_Constutional_Court_Decisions` (licence: CC-BY-4.0)

It contains Turkish Constitutional Court individual-application decisions with a binary
rights-violation label. The scripts and notebooks in this repository download it directly.

The specific 60 test instances used for the explanation analysis, together with their
stratum, gold label, model prediction, and prediction confidence, are listed in
`../results/sample_ids.csv`.
